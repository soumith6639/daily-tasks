Emmets

In the context of HTML and web development, "emmet" refers to a popular toolkit used to improve the efficiency of writing HTML and CSS code. Emmet allows developers to write short CSS-like selectors that are then expanded into full HTML code. This can significantly speed up the coding process.


Emmet
Usage
! or html:5 used to give html structure
tag name to create tags
> used to create child
{} used to represent content
+ used to create sibling
^ used to create parent to the element
